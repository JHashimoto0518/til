
# GitHubで開発をはじめるには

1. GitHubでリポジトリを作成する

2. リポジトリのURLをコピーする

3. ローカルにクローンする

    ```bash
    git clone https://github.com/xxx/yyy.git
    ```

4. コンフィギュレーション

    ```bash
    # 草が生えるようにする
    git config user.email "01233456790+xxx@users.noreply.github.com"
    git config user.name "Junichi Hashimoto"
    ```
