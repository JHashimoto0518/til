# 作業ディレクトリの変更を退避するには

stashエントリに保存する。

```bash
git stash push
# Saved working directory and index state WIP on master: 6e79f54 docs: READMEを修正
```

stashが作成される。

```bash
git stash list
# stash@{0}: WIP on master: 6e79f54 docs: READMEを修正
```

変更を復元する。

```bash
git stash pop
# ...
# Dropped refs/stash@{0} (9da4bxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx)
```
