# クエリを診断するには

Power BI Desktop であれば、クエリを診断できる。

インストーラーでセットアップできる。

https://www.microsoft.com/ja-jp/download/details.aspx?id=58494

## 診断の種類

診断には 3 つの種類がある。

1 つ目は、詳細ビューと概要ビューを備えたプライマリ診断。 概要ビューは、クエリで時間がかかっている場所をすぐに把握できるようにすることを目的としている。 詳細ビューは 1 行ずつのより詳細な情報。
