
# GitHubで開発をはじめるには

1. GitHub でリポジトリを作成する

2. リポジトリの URL をコピーする

3. ローカルにクローンする

    ```bash
    git clone https://github.com/xxx/yyy.git
    ```

4. コンフィギュレーション

    ```bash
    # 草が生えるようにする
    git config user.email "01233456790+xxx@users.noreply.github.com"
    git config user.name "my name"
    ```
